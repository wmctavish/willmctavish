PROJECTS

Submissions

Submissions is a training journal for BJJ, built around NodeJS and MongoDB. It uses a custom REST API and standard CRUD functionality. It also contains a video library, organized by concepts/positions. Goals can be added and marked as "completed", then displayed permanently.

StockDash

StockDash is a stock trading/monitoring dashboard built with React. It makes use of the Alpha Vantage API to query current stock data. ChartJS is used to display bogus portfolio data, and the "Trade" portion allows for simultaneous mock processing of stock sales and purchases. The "Market" portion gives a more complete dataset, as well as a search tool to find stock symbols.

Wedding Site

A personal site created for our wedding in September 2019. Photo gallery created with the Lightbox2 library. RSVP form gets posted through Formspree to capture responses.

Fancy Login

Woah, who's this guy? Very Fancy.

A strictly front-end registration component. Uses a mix of jQuery and vanilla JS to provide real-time validation of input fields based on their respective rulesets. Form submission button is made available only after succesful validation of all three fields. A tasteful amount of visual sugar for good mix.

Synth

A feature-rich monophonic synthesizer built around the ToneJS library. A looping step sequencer allows the user to alter parameters while listening to current sequence, as well as change note selection on the fly. Light/dark theme switching, sequence start/stop, and visual feedback of squence position make this browser synth one that you can really lose track of time with.

*Currently implementing cross-browser compatibility - only runs on Chrome, for now*

Song Selector

A simple means to keep track of all the songs I can play on guitar. Makes AJAX calls to a central JSON file.
